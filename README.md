

# Weather App

- Fullscreen mode
- Cross platform


## Setup

Clone the repository

```bash
  git clone https://github.com/Tofazzal-2443/weather_app.git
```
Move to the desired folder

```bash
  cd \weather_app
```
To run the app, simply write

```bash
  flutter pub get
```
```bash
  flutter run
```
